CONNECTION:
  connect                Connect to server (localhost:8080)
  disconnect             Disconnect from server
  exit                   Exit program
  help                   Show help

BASIC COMMANDS:
  create <name>                      Create vertex
  type <name> <type>                 Set vertex type
  parent <name> <parent>             Set parent (inheritance)
  ext <name> <extension> [type]      Add extension
  iface <name> <interface> <type>    Add interface

QUERY COMMANDS:
  query-start                        Start query mode
  query-querier <name>               Set querier vertex
  query-iface <name>                 Set interface vertex
  query-end <ok|failed|cached>       End query with result
  query-clear                        Clear all highlights

NODE TYPES:
  unknown                Gray     (default)
  component              Blue     (class)
  interface              Purple   (interface)
  tie                    Cyan     (tie type)
  boa                    Indigo   (boa type)
  data-extension         Green    (data ext)
  code-extension         Green    (code ext)
  transient-extension    Green    (transient ext)
  cache-extension        Green    (cache ext)

EDGE TYPES:
  Extension:     data, code, cache, transient
  Interface:     tie, tie-chain, boa
